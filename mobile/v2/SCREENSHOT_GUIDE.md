## How to Take Screenshots on iPhone

### Method 1: Physical Buttons
- **iPhone with Face ID**: Press Side button + Volume Up simultaneously
- **iPhone with Home button**: Press Side button + Home button simultaneously

### Method 2: AssistiveTouch
1. Settings ‚Üí Accessibility ‚Üí Touch ‚Üí AssistiveTouch ‚Üí On
2. Tap the floating button ‚Üí Device ‚Üí More ‚Üí Screenshot

### Method 3: Back Tap (iOS 14+)
1. Settings ‚Üí Accessibility ‚Üí Touch ‚Üí Back Tap
2. Set Double Tap or Triple Tap to "Screenshot"
3. Tap back of phone to capture

## Recommended Screenshot Content

### 1. Landing Screen
**What to show:**
- Full landing screen with logo
- Both action buttons visible ("Need token" / "Have token")
- Background gradient visible

**Tips:**
- Make sure status bar time looks normal (not 9:41 for authenticity)
- Full battery icon preferred

---

### 2. Token Generation (Mint) Screen
**What to show:**
- Validity dropdown (show "1 week" selected)
- Session time buttons (show "60m" selected)
- Message limit field
- "Generate token" button

**Tips:**
- Fill in realistic values
- Show all form elements

---

### 3. Token Display
**What to show:**
- Generated 12-character token clearly visible
- Copy/share functionality visible
- Countdown or validity info

**Tips:**
- Use a realistic-looking token like "DEMO1234WXYZ"

---

### 4. Token Entry (Accept) Screen
**What to show:**
- Token input field
- "Enter session" button
- Helpful text/instructions

**Tips:**
- Can show empty field with placeholder OR
- Partially typed token for action feel

---

### 5. Active Chat Session
**What to show:**
- Multiple messages exchanged
- Message bubbles (sent and received)
- Session timer visible
- Chat input field

**Sample conversation:**
```
You: Hey! Ready for our call?
Partner: Yes! The connection is great
You: Love how secure this is
Partner: And messages disappear after! üîí
```

**Tips:**
- Use natural, friendly conversation
- Show 4-6 messages
- Don't show anything inappropriate

---

### 6. Video Call
**What to show:**
- Both video feeds visible
- Call controls (mute, speaker, end call)
- Clean, professional appearance

**Tips:**
- Ensure good lighting for video
- Both participants should look engaged
- Consider blurring faces for privacy

---

### 7. Menu (Optional)
**What to show:**
- Open hamburger menu
- Legal links visible (Privacy Policy, Terms, Help)

## Transferring Screenshots

### From iPhone to Mac

**Method 1: AirDrop**
1. Select screenshots in Photos
2. Tap Share ‚Üí AirDrop ‚Üí Your Mac

**Method 2: iCloud Photos**
- Screenshots sync automatically if enabled

**Method 3: USB Cable**
1. Connect iPhone to Mac
2. Open Image Capture app
3. Select and import screenshots

## App Store Upload

1. Go to [App Store Connect](https://appstoreconnect.apple.com)
2. Select your app ‚Üí App Information ‚Üí Screenshots
3. Upload screenshots for each device size
4. Arrange in desired order (first screenshot is most important)
5. Add optional promotional text overlay using App Store Connect's tools
